1. препроцесс
   файл примера преобразуется в стандартный код на С с включениием
библиотеки "Wiring" и необходимых заголовков функций

2. компиляция
   файл из первого этапа компилируется avr-gcc, компилятором 
GNU для микропроцессора AVR

3. линковка
   скомпилированный файл линкуется, и создается стандартный файл
ELF

4. преобразования
   файл ELF преобразуется в формат 'Intel HEX'

5. загрузка
   файл в шестнадцатеричном формате отправляется в микросхему
через avrdude
